# TTT Decrease Train/Test Distribution Shift Numeric Analysis

口径：沿用 `ttt_decrease` 的 26 个下降数据集和 t-SNE 图的同一采样逻辑；在标准化数值特征 + one-hot 类别特征空间中，对每个真实类别分别比较 train/test。

指标：`domain_auc` 表示只用特征区分 train/test 的交叉验证 AUC，越接近 1 偏移越强；`norm_centroid_gap` 是同类别 train/test 中心距离除以类内半径；`label_tvd` 是整体类别比例差异。
